# 돈 정산 계산기

그룹 지출 정산을 위한 직관적인 웹 애플리케이션입니다.

## 설명

돈 정산 계산기는 여러 사람이 함께 사용하는 지출 정산을 쉽고 빠르게 처리할 수 있도록 도와주는 웹 서비스입니다. 복잡한 계산 과정을 자동화하여 누구나 쉽게 정산할 수 있습니다.

## 기능

### 참여자 관리
- 실시간 참여자 추가/제거
- 참여자 목록 자동 업데이트

### 지출 관리
- 직관적인 지출 항목 등록
- 지불자 지정 및 제외 인원 설정
- 실시간 금액 계산

### 정산 결과
- 개인별 정산 금액 자동 계산
- 최적의 정산 방법 제시
- 상세 정산 내역 확인

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- 반응형 웹 디자인
